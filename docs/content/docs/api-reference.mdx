## CosmioContainer Methods

| Method | Signature | Description |
|--------|-----------|-------------|
| `create` | `(doc) => Promise<Document>` | Create with Zod validation and defaults |
| `upsert` | `(doc) => Promise<Document>` | Upsert with Zod validation and defaults |
| `findById` | `(id, pk) => Promise<Document \| undefined>` | Point read by ID + partition key |
| `replace` | `(id, doc, opts?) => Promise<Document>` | Full document replacement (optional ETag) |
| `delete` | `(id, pk) => Promise<void>` | Delete (soft delete if configured) |
| `hardDelete` | `(id, pk) => Promise<void>` | Physical delete (bypasses soft delete) |
| `restore` | `(id, pk) => Promise<Document \| undefined>` | Restore a soft-deleted document |
| `patch` | `(id, pk, ops) => Promise<Document>` | Partial update with patch operations |
| `query` | `(spec, pk?) => Promise<Document[]>` | Raw SQL query |
| `find` | `(pk?) => QueryBuilder` | Start a fluent query builder |
| `findWithDeleted` | `(pk?) => QueryBuilder` | Query including soft-deleted docs |
| `bulk` | `(ops) => Promise<void>` | Execute bulk create/upsert/delete |
| `use` | `(event, fn) => this` | Register a lifecycle hook |
| `scope` | `() => CosmioContainer` | Create a request-scoped cached instance |
| `raw` | `Container` | Access underlying Cosmos DB Container |
| `model` | `ModelDefinition` | The bound model definition |
