## CosmioContainer メソッド

| メソッド | シグネチャ | 説明 |
|---------|-----------|------|
| `create` | `(doc) => Promise<Document>` | Zodバリデーションとデフォルト値付きで作成 |
| `upsert` | `(doc) => Promise<Document>` | Zodバリデーションとデフォルト値付きでアップサート |
| `findById` | `(id, pk) => Promise<Document \| undefined>` | ID + パーティションキーでポイントリード |
| `replace` | `(id, doc, opts?) => Promise<Document>` | ドキュメント全体の置換（オプションでETag） |
| `delete` | `(id, pk) => Promise<void>` | 削除（設定済みの場合はソフトデリート） |
| `hardDelete` | `(id, pk) => Promise<void>` | 物理削除（ソフトデリートをバイパス） |
| `restore` | `(id, pk) => Promise<Document \| undefined>` | ソフトデリートされたドキュメントを復元 |
| `patch` | `(id, pk, ops) => Promise<Document>` | パッチ操作による部分更新 |
| `query` | `(spec, pk?) => Promise<Document[]>` | 生SQLクエリ |
| `find` | `(pk?) => QueryBuilder` | フルエントクエリビルダーを開始 |
| `findWithDeleted` | `(pk?) => QueryBuilder` | ソフトデリート済みを含むクエリ |
| `bulk` | `(ops) => Promise<void>` | バルク作成/アップサート/削除を実行 |
| `use` | `(event, fn) => this` | ライフサイクルフックを登録 |
| `scope` | `() => CosmioContainer` | リクエストスコープのキャッシュ済みインスタンスを作成 |
| `raw` | `Container` | 基盤のCosmos DB Containerにアクセス |
| `model` | `ModelDefinition` | バインドされたモデル定義 |
