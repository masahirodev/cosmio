---
title: クエリビルダー
description: 型安全な Cosmos DB クエリを構築するためのフルーエント API
---

クエリビルダーは、完全な型安全性を備えた Cosmos DB SQL クエリを構築するためのフルーエントでチェーン可能な API を提供します。

## クエリの開始

```ts
const users = client.model(UserModel);

// パーティションキーでスコープ（パフォーマンス推奨）
const results = await users.find(["acme"]).exec();

// クロスパーティションクエリ（パーティションキーなし）
const allUsers = await users.find().exec();
```

## フルーエント API

### where()

フィルタ条件を追加します。[3つのスタイル](/ja/docs/queries/where-filters)をサポート:

```ts
users.find(["acme"])
  .where({ name: { contains: "Ali" }, role: "admin" })
  .exec();
```

### orderBy()

```ts
users.find(["acme"])
  .orderBy("createdAt", "DESC")
  .orderBy("name", "ASC")
  .exec();
```

### limit()

```ts
users.find(["acme"])
  .limit(20)
  .exec();
```

### offset()

```ts
users.find(["acme"])
  .offset(20)
  .limit(20)
  .exec();
```

## 完全な例

```ts
const results = await users
  .find(["acme"])
  .where({
    role: "member",
    createdAt: { gte: 1700000000 },
    name: { startsWith: "A" },
  })
  .orderBy("createdAt", "DESC")
  .limit(10)
  .offset(0)
  .exec();
```

## 生成されたクエリの確認

`toQuerySpec()` を使って Cosmos DB に送信される SQL を確認できます:

```ts
const builder = users
  .find(["acme"])
  .where({ role: "admin" })
  .orderBy("createdAt", "DESC")
  .limit(5);

console.log(builder.toQuerySpec());
// {
//   query: "SELECT TOP 5 * FROM c WHERE c.role = @p0 ORDER BY c.createdAt DESC",
//   parameters: [{ name: "@p0", value: "admin" }]
// }
```

## whereRaw() との組み合わせ

型付きフィルタと Raw SQL を組み合わせて高度な式を作成:

```ts
const results = await users
  .find(["acme"])
  .where({ role: "member" })
  .whereRaw("ARRAY_LENGTH(c.tags) > @minTags", { "@minTags": 2 })
  .exec();
```
